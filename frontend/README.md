# Confiarei Frontend

Esse projeto é feito com [Angular CLI](https://github.com/angular/angular-cli) versão 13.3.3.

## Tecnologias usadas

- Angular
- Material Design
- Sass

## Rodando localmente

- Rode `npm install`
- Instale o CLI do angular: `npm install -g @angular/cli`
- Após isso, rode: `ng serve`
